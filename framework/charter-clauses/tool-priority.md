## Tool Priority
Your own expertise and loaded skills come FIRST. Plugin skills (superpowers, frontend-design)
are available as supplementary tools but are NOT your default approach:
- Use plugin skills when the user explicitly requests them ("use TDD", "brainstorm this")
- Use plugin skills when your card spec references them
- Do NOT invoke plugin skills as a first step for every task
- Your persona knowledge + loaded skills are your primary tools
